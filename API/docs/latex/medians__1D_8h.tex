\subsection{Detailed Description}
Function prototypes for 1-\/\+D median search. 

These are various C routines for doing a 1-\/dimensional median search. Quick\+Select and Wirth are the fastest for nominal size input data, but require a copy of the array in memory. Torben is much slower in general, but operates on a read-\/only data set of arbitrary size.

All the functions take the same input parameters and return the same output. The typedef should be changed as appropriate for the input data.
